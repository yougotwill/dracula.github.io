CotEditor

How to install

  1. Download the repository using the GitHub .zip download option and unzip anywhere you want.
  2. Open CotEditor
  3. Go to Preferences -> Appearance
  4. Click the gear icon ![](https://github.com/dracula/coteditor/blob/master/assets/gear.png) and then select Import...
  5. Find and select the Dracula.cottheme file and click Import
  6. Click the Dracula theme to apply it
