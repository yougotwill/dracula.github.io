Brackets

Install using Brackets

  1. Go to File -> Extension Manager...
  2. Switch to Themes tab
  3. Search for Dracula Official
  4. Click Install

Activating theme

Go to Preferences... and add "themes.theme": "dracula" to the brackets.json file.
